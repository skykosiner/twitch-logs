# Tlogs

* This is a simple cli tool to allow to view twitch logs for a user in a
  channel

## Install
```
git clone https://github.com/skykosiner/twitch-logs
cd twitch-logs
cd ./cmd/tlogs/
go install .
```

## Usage
* It's quite simple you just need to run the tool and pass in the channel and
  username you want `tlogs [channel] [username]`
